# imagine-diagrams

Use this skill for creating clean, consistent **pure-SVG** diagrams in Imagine.

## Workflow

1. Start from a scaffolded diagram (`imagine-scaffold` kinds: `diagram-pipeline`, `diagram-architecture`).
2. Define a grid and consistent spacing (x/y gaps, box sizes).
3. Use primitives (`Box`, `Arrow`, `Label`, `Callout`) and theme tokens.
4. Keep labels short and align baselines.
