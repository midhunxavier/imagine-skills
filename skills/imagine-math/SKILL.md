# imagine-math

Use this skill for paper-ready math and equation figures in Imagine.

## Workflow

1. Start from `imagine-scaffold` kind `math-equation`.
2. Prefer MathJax SVG output (vector-first).
3. Keep equation size readable at final print size; avoid overly long lines.
